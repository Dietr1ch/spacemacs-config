#+title: My Spacemacs configuration

* Info
This is my personal [[https://www.spacemacs.org/][spacemacs]] configuration.

* Setup
This is a simplified setup overview. Might require some backups and ~mkdir -p~.

** ~chemacs2~
Clone,
#+begin_src fish
git clone \
    --origin 'upstream' \
    --single-branch \
    https://github.com/plexus/chemacs2.git \
    ~/.emacs.d
#+end_src

Configure,
#+begin_src fish
  echo -n '
  (
   ;; Spacemacs @develop
   ("spacemacs" . (
                   (user-emacs-directory . "~/.config/spacemacs/distro")
                   (env . (
                           ("SPACEMACSDIR" . "~/.config/spacemacs/config")
                           ))
                   ))
   )
  ' >~/.emacs-profiles.el

  echo spacemacs >~/.emacs-profile
#+end_src

** ~spacemacs~

git clone \
